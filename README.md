# R.A.A Education

Platform edukasi berbasis AI untuk mendukung pengalaman belajar yang lebih cerdas dan personal.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   ```bash
   npm run dev
   ```
